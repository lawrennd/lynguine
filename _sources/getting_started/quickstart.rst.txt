Quick Start Guide
=================

Get started with ndlpy quickly with this brief guide.

Creating Your First ndlpy Object
--------------------------------

Here's an example of how to create and use a basic ndlpy object:

.. code-block:: python

    import ndlpy

    # Example code to create an object
    obj = ndlpy.YourClass()
    obj.method()
